QGIS Mascaret Plugin

Mascaret Plugin is a plugin for QGIS which allows you to build a 1D river model for Mascaret Software. Mascaret software is a module of the Telemac-Mascaret system dedicated to 1D river modeling. More informations could be found on Mascaret website

Installation

Use the QGIS Plugins menu to install the Mascaret Plugin
